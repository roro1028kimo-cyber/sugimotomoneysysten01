# 101Form 支出證明單設計分析

## 一、版面結構
- **尺寸**：A4 直式
- **標題**：支出證明單 (置中，大字)
- **表頭資訊**：
  - 紀錄編號 (右上)
  - 申請日期 (右上)
- **表格欄位**：
  - 申請人 | 申請部門
  - 科目 (會計科目)
  - 事由 (支出原因)
  - 不能取得單據原因 (重點欄位)
  - 實付金額 (新台幣)
  - 備註
- **簽核區塊**：
  - 位於表格下方，包含：申請人、部門主管、會計、核准

## 二、與現有系統的對應
- **申請人**：對應系統中的 `contactPerson` (或新增欄位)
- **申請部門**：目前無此欄位，可先留空或預設
- **科目**：對應 `description` 中的會計科目備註
- **事由**：對應 `description`
- **不能取得單據原因**：這是支出證明單的特性，通常用於無發票核銷。我們可以在列印時預設為「遺失」或「廠商免用統一發票」，或留空手寫。
- **實付金額**：對應 `amount`

## 三、設計調整計畫
1.  **尺寸**：由 B4 橫式改回 **A4 直式**。
2.  **標題**：移除「杉本公司」大字，改為簡潔的「支出證明單」。
3.  **表格樣式**：
    - 欄位名稱靠左或置中，背景色為淺灰或白色。
    - 線條細緻，強調填寫空間。
4.  **新增欄位**：
    - 加入「不能取得單據原因」欄位，符合支出證明單的用途。
