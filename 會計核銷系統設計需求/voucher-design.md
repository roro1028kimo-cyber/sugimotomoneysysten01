# 杉本公司會計憑證設計規劃

## 一、設計理念
基於杉本公司的品牌特性：
- **直覺式視覺**：一目了然，無多餘裝飾。
- **功能至上**：欄位配置符合會計核銷邏輯。
- **黑白色調**：專業、簡約、節省列印墨水。
- **邏輯性**：資訊層級分明（誰付錢 -> 付給誰 -> 多少錢 -> 為什麼 -> 誰核准）。

從搜尋結果中，我們可以看到幾種常見格式：

## 三、杉本專屬憑證設計 (A4 直式，一頁兩張或一張)

### 3.1 版面配置 (Layout)
採用 **A5 橫式** 尺寸設計（A4 紙張可印兩張，或置中印一張），方便裁切歸檔。

### 3.2 核心欄位
| 區塊 | 內容 | 設計風格 |
|------|------|----------|
| **Header** | 公司 Logo (左)、單據名稱「支出證明單」(中)、憑證編號/日期 (右) | 黑體大字，強調權威感 |
| **Payee** | 受款人/廠商名稱、統一編號 | 粗框線強調 |
| **Amount** | 金額 (新台幣)、大寫金額 | 獨立區塊，數字特大 |
| **Details** | 摘要/用途說明、會計科目 (選填) | 足夠的書寫/列印空間 |
| **Payment** | 付款方式 (現金/匯款/支票)、匯款帳號 | 勾選框設計 |
| **Footer** | 經手人、部門主管、會計、核准 | 簽名欄位底部置中 |

### 3.3 視覺元素
- **線條**：使用 1px 或 2px 的黑色實線，不使用虛線（避免列印模糊）。
- **字體**：
  - 標題：Noto Sans TC (Bold)
  - 內文：Noto Sans TC (Regular)
  - 數字：Roboto Mono (等寬字體，方便對齊)
- **Logo**：置於左上角，黑白版本。

## 四、PDF 實作技術
使用 `react-to-print` 或直接透過 CSS `@media print` 進行樣式控制，這是最輕量且直覺的做法。
- 隱藏網頁導航與按鈕。
- 設定 `@page { size: A4; margin: 2cm; }`。
- 使用 CSS Grid 進行精準排版。
